## tools/plot_training_curves.py

### 功能

读取某次训练目录中的 `history.csv`，生成可视化图片 `training_curves.png`，用于快速观察训练过程：

1. 训练损失（Train Loss）
2. 训练准确率（Train Acc）
3. 验证/评估分数（Eval Score）

脚本会自动标注 Eval Score 的最佳点（Best）。

### 输入参数

- `--input`（必填）
  - 训练目录路径，或直接传 `history.csv` 文件路径。
- `--output`（可选）
  - 输出图片路径；不填时默认保存到训练目录下的 `training_curves.png`。

### 输出文件

- 默认输出：`<run_dir>/training_curves.png`

### 使用示例

1. 传入训练目录：

```bash
python tools/plot_training_curves.py --input "logs/full_medbiasx-20260319-205907"
```

2. 直接传入 history.csv：

```bash
python tools/plot_training_curves.py --input "logs/full_medbiasx-20260319-205907/history.csv"
```

3. 指定输出路径：

```bash
python tools/plot_training_curves.py \
  --input "logs/full_medbiasx-20260319-205907/history.csv" \
  --output "logs/full_medbiasx-20260319-205907/my_training_curves.png"
```

### 注意事项

1. 需要先完成训练并生成 `history.csv`。
2. 若缺少依赖，请先安装：

```bash
pip install matplotlib
```

3. 若 `history.csv` 为空，脚本会报错并提示对应路径。
